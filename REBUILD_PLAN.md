# LSE GALLERY — 설계 노트

승은(Seung Eun)의 유화 195점을 월별 타임라인으로 보여주는 사이트와, 콘텐츠를 로컬에서
편집하는 관리자. 프레임워크·CDN 없이 바닐라 JS/CSS 로 작성했고, 데이터는 `content.json`
하나뿐이다.

## 데이터 모델 (`content.json`)
```
{
  "meta": {"brand": "Seung Eun", "title": "LSE GALLERY"},
  "slides": [
    { "id": "s0", "image": "/images/x.png",
      "top": "1",                     // 순번(배열 순서와 함께 관리자에서 재부여)
      "bottom": "Inspire (January)",  // 카테고리 + 월 라벨
      "category": "Inspire",
      "title": "작품명", "desc": "설명(문단 \n 구분)",
      "date": "2026-01-01",           // 연-월. 사이트 정렬·눈금·상세 DATE 의 근거
      "size": "65.1 × 53.0 cm (15호)", "ptype": "Oil on canvas",
      "media": [ {"type":"image","src":"/images/a.png","caption":"","pos":0,"align":"center"} ]
    }
  ],
  "wordmark": {"l1": "LSE GALLERY", "l2": "", "l3": "-"},
  "texts": {"/about": {"title": "...", "intro": "...", "thanks": "..."}}
}
```
- 작품 순서는 **`date` 내림차순**(최신 우선). 같은 날짜면 배열 순서를 유지한다.
- 월 그룹(눈금·인덱스 라벨·인트로 목록)은 `bottom` 의 `(Month)` + `date` 의 연도로 만든다.

## 뷰
- **타임라인** `/` — 퍼-슬라이드 wrap 무한 캐러셀(트랙 전체를 옮기지 않는다) + 캔버스 눈금.
- **Surf** `/surf` — 3D 카드 덱. 사인 부유 + 각도가 열리며 등장.
- **인덱스** `/articles` — 12열 그리드, 스크롤 속도에 따라 행이 기운다.
- **상세** `/p/<id>` — 오버레이. 클릭한 그림이 슬롯에서 상세로 **날아가고**, 닫으면 되돌아온다.
- **About** `/about` — 커튼 페이지 + 아이콘 크레딧.

## 뷰 간 공유 플립 (핵심 메커니즘)
슬롯(`.lse-slot`, 클리핑 없음) 안에 프레임(`.lse-frame`, 클리핑)이 있고, **프레임이 뷰 사이를
실제로 이동**한다. 복제본을 만들지 않는다 — 같은 픽셀이 움직인다.

1. 떠나는 뷰가 보고 있던 작품의 인덱스를 `body.dataset.index` 로 넘긴다.
2. 도착 뷰는 그 작품으로 먼저 점프한 뒤 "준비됨"을 알린다(최대 200ms 대기).
3. 두 뷰의 `data-id` 를 교집합으로 맞춘다 — **화면에 보이는 것만** 대상(+ 캐러셀 중앙 슬라이드).
4. 짝이 맞은 프레임은 리파런팅 후 이전 위치에서 날아온다. 짝이 없는 슬롯은 아래에서 올라온다.
5. surf 를 떠날 때만 예외: 플립 대신 그림들이 위로 빠져나간다.

## 하드 룰 (어기면 재발했던 버그)
1. 시각 이슈는 **헤드리스 프레임 캡처로 검증** — 추측 수정 금지. selenium + 격리 프로필.
   (`tools/verify_flip.py`, `tools/verify_exit.py`)
2. 플립은 복제 금지, 실제 프레임 리파런팅. 캐러셀 `goTo` 는 상세를 **닫는 시점에만**.
3. rect 측정은 DOM 삽입 후 rAF (fragment 상태면 0-rect → NaN transform 이 조용히 무시된다).
4. 트랙 전체 translate 금지(거대 레이어). `will-change` 대량 금지 — 카드마다 걸면 deck 의 3D
   깊이 정렬에서 빠져 겹침 순서가 뒤집힌다.
5. 뷰가 떠날 때는 **진행 중이던 도착 애니메이션을 그 자리에서 정지**시킨다. 그러지 않으면
   화면 밖 카드가 뒤늦게 올라왔다가 뷰 제거와 함께 툭 사라진다.
6. 착지 후 인라인 스타일 정리는 **아직 현재 뷰에 있는 요소만** 건드린다.
